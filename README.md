# CartPole-Visual-DQN
یک برنامه آموزشی که با استفاده از یادگیری تقویتی (Reinforcement Learning) میله‌ی CartPole را روی صفحه کنترل و آموزش می‌دهد و وضعیت محیط را به صورت تصویری نمایش می‌دهد.

## ⚡ ویژگی‌ها

🖥️ نمایش محیط CartPole به صورت پنجره زنده با render_mode="human"

🧠 شبکه DQN ساده برای تصمیم‌گیری هوشمند بر اساس وضعیت محیط

🔄 استفاده از Replay Buffer برای بهبود آموزش

📈 بهینه‌سازی با epsilon-greedy و decay پویا

💾 ذخیره حافظه‌ی تجربه برای یادگیری پایدار

🚀 اجرای سریع روی CPU و امکان استفاده از GPU

## 🛠️ تکنولوژی و کتابخانه‌ها

Python 3.13+

PyTorch برای شبکه عصبی

Gymnasium برای محیط RL

collections.deque برای Replay Buffer

random و numpy برای عملیات تصادفی و عددی

## ⚙️ نحوه کار

محیط CartPole ایجاد می‌شود و روی صفحه به شما نمایش داده می‌شود.

Agent با استفاده از شبکه DQN اقدام به انتخاب اکشن می‌کند.

تجربه‌ها در Replay Buffer ذخیره می‌شوند و شبکه آموزش می‌بیند.

بعد از چند اپیزود، Agent می‌تواند میله را به صورت پایدار روی گاری نگه دارد.
